- 날짜 및 시간 확인 : `date`
- 시스템 정보 확인
  - `hostname`
  - `uname`
    - 옵션 -a
- 사용자 정보 확인
  - 사용자 목록 : `who`
  - 현재 사용자 : `whoami`
- 디렉터리 내용 확인 : `ls`
- 패스워드 변경 : `passwd`
- 화면 정리 : `clear`
- 온라인 매뉴얼 : `man`
- 명령어에 대한 간단한 설명 : `whatis`
