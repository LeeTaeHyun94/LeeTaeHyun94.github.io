## 1. 프로그래밍 언어란?

- 프로그래밍 언어 : 동작시키는 프로그램을 작성하기 위한 인공적인 언어
- 프로그래밍 언어의 분류 : 사용하기 편리한 정도에 따라 분류
  - 저급 언어 : 하드웨어 지향의 기계 중심 언어, 하드웨어와 밀접한 기능 제어
    - 기계어, 어셈블리어
  - 고급 언어
    - 컴퓨터 기종에 따라 다르게 표현되는 저급 언어의 문제점 해결
    - 사람이 사용하는 기호 체계와 유사
    - C, C++, Java 등 대부분의 언어
